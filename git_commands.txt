1. Git Add:
   used for: Adding files to Staging Area
   cmd: git add <file_name(s) OR "." for ALL>

2. Git Commit:
   used for: Commiting files with proper message (-m)
   cmd: git commit -m "<Your message>"

3. Git Push:
   used for: Pushing changes to remote branch
   cmd: git push origin master
   explaination: 
        i.  origin: local active branch name
        ii. master: Remote branch name
4. Git diff:
   used for: to check file difference
   cmd: git diff <File name>

5. Git fetch:
    used for: to see new branched which are created in remote
    cmd: git fetch

6. Git Status
   used for: To see the files list that have changed
   cmd: git status
